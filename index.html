Portifólio

Durante meus cursos de desenvolvimento web, design UX/UI e JavaScript, desenvolvi alguns sites, para ter experiências ou para ajudar amigos que gostariam de criar sua própria vitrine virtual.

  • Desenvolvi este site para um doutorando na Itália, criando desde a identidade visual no Figma (logo, ilustrações e paleta) até a implementação com HTML, CSS e JavaScript. O projeto inclui slides de artigos e menu mobile responsivo, aplicando conceitos de design limpo e código semântico.

  • Desenvolvi um site em JavaScript chamado "Animais Fantásticos", com informações, imagens e curiosidades sobre 6 animais. Usei HTML, CSS e JS para criar navegação dinâmica, abas de conteúdo e efeitos interativos. O projeto me ajudou a praticar manipulação do DOM e responsividade.

  • Desenvolvi o site da Bickraft no curso e UI & UX Design da Origamid, a idéia era criar um site de venda de bicicletas elétricas como todas as páginas de cada produto, com uma página de orçamento interativa, venda de seguros, entre outros. Este projeto foi desenvolvido com a ferramenta Figma, que possibilitou o planejamento, organização de layout e a criação de um protótipo com a ferramenta Autoflow.

  • O Dogs foi desenvolvido nos cursos de React e Next.js da Origamid, como uma rede social para pets. Utilizei React para funcionalidades de cadastro, posts e comentários, e otimizei com Next.js para melhor performance, SEO e renderização no servidor, integrando gráficos de visualizações via bibliotecas externas.

  • O FlexBlog foi o projeto prático do curso CSS Flexbox, onde apliquei todos os fundamentos para construir um blog 100% responsivo com Flexbox. O desafio principal foi organizar os elementos (menu, posts, sidebar, footer) de forma dinâmica, garantindo adaptação em diferentes telas sem complicações.

  • O Slide Stories foi desenvolvido no curso de TypeScript da Origamid como um sistema de slides interativo. Utilizei TypeScript puro para manipulação avançada do DOM, controlando navegação e temporização. Implementei type guards e interfaces para garantir segurança de dados durante as interações. O projeto solidificou meus conhecimentos em tipagem estática e JavaScript moderno.
